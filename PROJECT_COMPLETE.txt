🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════════

OPTION 1: DOCKER COMPOSE (Recommended - Easiest)
────────────────────────────────────────────────
1. cd "d:\My Data\Coding\SAHIFALAB\Telegram App"
2. copy backend\.env.example backend\.env
3. copy frontend\.env.example frontend\.env
4. copy bot\.env.example bot\.env
5. Add your Telegram Bot token to all .env files
6. docker-compose up -d

Access:
  - Frontend: http://localhost:3000
  - API Docs: http://localhost:8000/docs
  - Database: localhost:5432

OPTION 2: MANUAL SETUP (3 Terminals)
───────────────────────────────────
Terminal 1 - Backend:
  cd backend
  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements.txt
  uvicorn app.main:app --reload

Terminal 2 - Frontend:
  cd frontend
  npm install
  npm run dev

Terminal 3 - Bot:
  cd bot
  python -m venv venv
  venv\Scripts\activate
  pip install -r requirements.txt
  python main.py
